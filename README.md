# 🌸 CharOS

> **An open-source AI desktop companion platform.**

CharOS is a local-first, modular framework for building persistent AI
desktop companions. The default companion is **Nila**, but the platform
is designed so anyone can create their own characters, personalities,
voices, and skills.

------------------------------------------------------------------------

## Vision

Most AI assistants are chat windows.

**CharOS aims to make AI feel like a living desktop companion** that
can:

-   🎤 Listen to voice commands
-   💬 Chat naturally
-   🧠 Remember important things
-   👁️ Understand your screen
-   💻 Write and fix code
-   ⚙️ Automate desktop tasks
-   🌸 Express itself with animations and personality

------------------------------------------------------------------------

## Core Features (Planned)

-   ✨ Side-docked transparent desktop companion
-   🎭 VRM character support
-   🎙️ Speech-to-text (Handy-Parakeet)
-   🧠 Local LLM orchestration (Ollama)
-   👀 Vision (Qwen2-VL)
-   💻 Coding agent (Qwen3 Coder Heretic)
-   📚 Obsidian-powered long-term memory
-   🕸️ Knowledge graph integration (Cognee)
-   🤖 Agentic planning & skills
-   🧩 Plugin system
-   🌐 Optional browser fallback for user-owned AI services
-   🔒 Local-first and privacy-aware

------------------------------------------------------------------------

# Project Philosophy

CharOS is **not** another chatbot.

It is a **desktop operating companion**.

The user interacts with a character---not with model names.

The platform automatically routes tasks to the appropriate capability.
